حدد هنا ترتيب المناصب من الأعلى سلطة إلى الأدنى (أو العكس)، وسيتم بناءً على هذا الترتيب تصعيد أو إحالة أي مراسلة أو بلاغ عبر المستويات الإدارية بالتسلسل الذي تحدده. تلقائياً ودون أي إعداد إضافي: كل منصب لا يحق له مخاطبة (تصعيد/إحالة) إلا المنصب الأعلى منه مباشرة والأدنى منه مباشرة في هذا الترتيب فقط — وليس أي منصب آخر.

قاعدة البيانات الحية (Firebase) — موصى بها

هذه الطريقة الموصى بها لمشاركة البيانات بين كل الأجهزة والحلقات: بمجرد ربطها، تتحدث كل الأجهزة تلقائياً وفورياً مع بعضها **بدون أي خطوة يدوية** — لا "مزامنة الآن"، ولا ربط لكل جهاز على حدة. أي موظف جديد يُضاف يظهر تلقائياً على كل الأجهزة خلال ثوانٍ، ويكفيه إدخال اسم المستخدم وكلمة السر مباشرة.

كيف أنشئ مشروع Firebase مجاني؟ (خطوات بالتفصيل، 5 دقائق)
  1. اذهب إلى console.firebase.google.com وسجّل الدخول بحساب Google.
  2. اضغط "Add project" وأنشئ مشروعاً جديداً بأي اسم (مثال: samail-quran).
  3. من القائمة اليسرى: Build ← Realtime Database ← Create Database ← اختر "Start in test mode" مبدئياً.
  4. من القائمة اليسرى أيضاً: Build ← Authentication ← Get started ← من تبويب "Sign-in method"، فعّل مزوّد "Anonymous" (هذه طبقة حماية إضافية مهمة، فعّلها حتى لو بدت غير ضرورية الآن).
  5. ارجع لتبويب "Rules" داخل Realtime Database، واستبدل المحتوى بـ:
    {"rules": {".read": "auth != null", ".write": "auth != null"}} ثم اضغط "Publish".
  6. ارجع لإعدادات المشروع (⚙️ Project settings) ← انزل لقسم "Your apps" ← اضغط أيقونة الويب </> لإنشاء تطبيق ويب جديد.
  7. انسخ الكائن الكامل الذي يظهر أمامك باسم firebaseConfig (يبدأ بـ const firebaseConfig = {...}) والصق **المحتوى بين الأقواس المعقوفة فقط** بالحقل أعلاه.

طبقة الحماية الإضافية: بقواعد "auth != null" أعلاه، لم يعد بإمكان أي طرف قراءة أو تعديل بياناتك عبر رابط قاعدة البيانات مباشرة (مثل الروبوتات والماسحات الآلية التي تبحث عن قواعد بيانات Firebase العامة على الإنترنت) — يجب المرور عبر تطبيقك أولاً، الذي يقوم بمصادقة مجهولة تلقائياً خلف الكواليس.

تنبيه أمني صادق (يبقى قائماً حتى مع هذه الطبقة): بما أن إعدادات Firebase مضمّنة داخل كود موقعك العلني، فإن شخصاً متمكناً تقنياً يقرأ الكود يمكنه نظرياً نسخ الإعدادات واستخدامها لإجراء مصادقة مجهولة خاصة به هو أيضاً، ثم الوصول لبياناتك. هذه الطبقة ترفع مستوى الحماية بشكل كبير وتوقف أغلب المحاولات الآلية العشوائية، لكنها ليست حماية مطلقة ضد شخص مستهدف تقنياً بشكل متعمّد. لحماية أقوى من ذلك، يتطلب الأمر بناء خادم خلفي حقيقي (Backend) — أخبرني إن رغبت بمناقشة ذلك مستقبلاً.

مزامنة مع Google Sheet (اختيارية — لعرض البيانات كجدول، أو كبديل عن Firebase)

عند ربط رابط "تطبيق ويب" (Web App URL)، تعمل المزامنة بالاتجاهين: كل تعديل تقوم به بالنظام (طلاب، موظفون، حلقات، برامج، سندات مالية، إنجازات، اختبارات، عهد، مراسلات) يُدفع فوراً للشيت، وبالمقابل يسحب النظام أي إضافة أو تعديل تقوم به مباشرة داخل الشيت نفسه ويدمجه بالنظام تلقائياً — لضمان دقة البيانات في الجهتين معاً.

كيف أحصل على رابط تطبيق الويب؟ (خطوات الربط بالتفصيل)
  1. أنشئ ملف Google Sheet جديد بأي اسم تريده (مثال: قاعدة بيانات مركز سمائل).
  2. من القائمة أعلى الشيت اختر: الإضافات (Extensions) ← Apps Script.
  3. احذف الكود الموجود، وألصق كود الربط الجاهز الظاهر أدناه بالكامل.
  4. اضغط على "نشر" (Deploy) ← "نشر جديد" (New deployment).
  5. اختر النوع "تطبيق ويب" (Web app)، وحدد صلاحية الوصول "Anyone" (أي شخص)، ثم اضغط Deploy.
  6. وافق على صلاحيات الوصول المطلوبة (هذه الموافقة تتم من حسابك أنت في Google ولا يمكن لأي طرف آخر القيام بها نيابة عنك)، ثم انسخ الرابط الناتج (Web app URL) وألصقه بالحقل أعلاه واضغط "حفظ الربط".
  7. ستُنشأ التبويبات المطلوبة تلقائياً بأول عملية حفظ من كل نوع — لا حاجة لإنشائها يدوياً.
  8. لتفعيل السحب من الشيت (المزامنة العكسية): فعّل خيار "السحب التلقائي الدوري" أعلاه، أو اضغط "مزامنة الآن يدوياً" في أي وقت.

للأمان: حذف صف من الشيت لا يحذف البيانات المقابلة من النظام تلقائياً (لتفادي فقد بيانات بالخطأ)، وتعديل تبويب الموظفين من الشيت يُحدِّث البيانات الأساسية فقط لموظفين موجودين فعلاً، ولا يُنشئ حسابات جديدة ولا يغيّر كلمات السر. 

function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var payload = JSON.parse(e.postData.contents);

  var sheetNames = {
    financial: "السندات المالية",
    students: "الطلاب",
    employees: "الموظفين",
    branches: "الحلقات والفروع",
    programs: "البرامج القرآنية",
    achievements: "إنجازات الحفظ",
    exams: "الاختبارات",
    assets: "العهد والأصول",
    complaints: "المراسلات والبلاغات"
  };

  var type = payload.type || "عام";
  var sheetName = sheetNames[type] || ("بيانات - " + type);
  var sheet = ss.getSheetByName(sheetName);
  if (!sheet) {
    sheet = ss.insertSheet(sheetName);
  }

  var data = payload.data || {};
  var headers = Object.keys(data);
  var idKey = headers[0];

  if (sheet.getLastRow() === 0) {
    sheet.appendRow(headers.concat(["الإجراء", "آخر تحديث"]));
  }

  var existingHeaders = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0];
  var idColIndex = existingHeaders.indexOf(idKey);
  var lastRow = sheet.getLastRow();
  var targetRow = -1;

  if (idColIndex > -1 && lastRow > 1) {
    var idValues = sheet.getRange(2, idColIndex + 1, lastRow - 1, 1).getValues();
    for (var i = 0; i < idValues.length; i++) {
      if (String(idValues[i][0]) === String(data[idKey])) {
        targetRow = i + 2;
        break;
      }
    }
  }

  var rowValues = existingHeaders.map(function (h) {
    if (h === "الإجراء") return payload.action || "إضافة";
    if (h === "آخر تحديث") return new Date().toLocaleString();
    var v = data[h];
    return (v === null || v === undefined) ? "" : v;
  });

  if (targetRow > -1) {
    sheet.getRange(targetRow, 1, 1, rowValues.length).setValues([rowValues]);
  } else {
    sheet.appendRow(rowValues);
  }

  return ContentService.createTextOutput(JSON.stringify({ status: "success" })).setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var result = {};
  var sheets = ss.getSheets();

  sheets.forEach(function (sheet) {
    var name = sheet.getName();
    if (sheet.getLastRow() < 1) { result[name] = []; return; }

    var values = sheet.getDataRange().getValues();
    var headers = values[0];
    var rows = [];

    for (var r = 1; r < values.length; r++) {
      var obj = {};
      for (var c = 0; c < headers.length; c++) {
        obj[headers[c]] = values[r][c];
      }
      rows.push(obj);
    }
    result[name] = rows;
  });

  return ContentService.createTextOutput(JSON.stringify(result)).setMimeType(ContentService.MimeType.JSON);
}
